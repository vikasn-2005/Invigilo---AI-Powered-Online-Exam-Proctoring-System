# 🎓 Invigilo – AI Powered Online Exam Proctoring System

## 📌 Overview
Invigilo is a full-stack smart exam proctoring system designed to conduct secure and efficient online examinations. It provides a controlled environment where administrators can create and manage exams, and students can attempt them seamlessly while maintaining exam integrity.

---

## 🚀 Features
- 👨‍🏫 Admin dashboard to create and manage exams  
- 📝 Support for MCQ and descriptive questions  
- 🔐 Secure authentication using JWT  
- 📊 Automatic evaluation for objective questions  
- ⚠️ Violation detection (tab switching, suspicious activity)  
- 📈 Result generation and performance tracking  

---

## 🛠️ Tech Stack

### 📱 Frontend
- Flutter (Dart)

### 🌐 Backend
- Node.js  
- Express.js  
- MongoDB
